# About

This is a simple personal financial management app, built with **Node.js + Express and MongoDB** to practice my backend skills.

# What's under the hood?
This project is up using:
* Node.js
* Express
* Typescript
* MongoDB
* Mongoose
* Husky
* Prettier
* ESLint
* Lint Staged
* Yarn

# How to run

Gotta have **npm** or **yarn** installed.
